## Features
- Glassmorphism UI with dark mode
- AI/technology background image
- Real-time validation for Name, Email, and Password
- Password strength indicator (Low / Medium / Strong)
- Show/Hide password using eye & eye-slash icons
- Submit button enabled only when all fields are valid
- Clear/Reset button
- Responsive design

## Technologies Used
- HTML5
- Tailwind CSS
- JavaScript (DOM Manipulation)

## Validation Rules
- Name field cannot be empty
- Email must be in valid format
- Password must be at least 6 characters
- Password strength updates in real-time

## How to Run
1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Fill in the form to see validation and password strength in action.
